## What Was Built or Changed
- `propedge-deploy/index.html` — 5 targeted patches to primary scoring function (`parseCSV`) and `calculateKellyWager`:
  1. **Cushion Score** — L5Avg vs line gap normalized as %, ±4–8 pts to modelScore for OVER/UNDER
  2. **Streak Regression** — 6+ game streaks (NBA/NHL) and 7+ (MLB) now carry a small penalty instead of bonus; actuarially correct mean reversion
  3. **Bayesian Shrinkage** — `trueProb` is now Bayesian-calibrated: `(n × hitRate + k × prior) / (n + k)` with sport-specific priors (NBA k=15, NHL k=12, MLB k=20), pulls thin-sample props toward 52% league baseline
  4. **Kelly → 1/3 fractional** — changed from half-Kelly (0.5×) to one-third (0.333×), hard cap at 5%, gate adjusted to 0.51 to handle shrunk probabilities
  5. **Game Total Correlation** — sourced from ESPN live odds (`competition.odds[0].overUnder`) already on every game object; z-score vs league avg adjusts volume props ±3–6 pts; falls back to sheet `total` column if ESPN data unavailable
