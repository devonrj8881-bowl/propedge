TROUBLESHOOTING QUICK FIXES
════════════════════════════

"No CSV found for NBA"
  → PropFinder UI changed
  → Check CSS selector in scraper line 507
  → May need to update download button identifier

"Retrieved NHL pace data for 0 skaters"
  → Natural Stat Trick website down or changed
  → Normal — will auto-recover on next run
  → Script continues without enrichment (not a blocker)

"Google Sheets write failed"
  → credentials.json expired
  → Regenerate OAuth: rm credentials.json (will prompt on next run)
  → Verify GOOGLE_SHEET_ID in .env is correct

"PropFinder login failed"
  → Credentials wrong in .env
  → Test manually at propfinder.com
  → Update .env with correct email/password

"Chrome not found"
  → Install Chrome or Chromium
  → Or update CHROME_PATH in scraper if at non-standard location
